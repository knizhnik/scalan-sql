Detailed Description

template<class O, class I, class K, void(*)(K &key, O const &outer) outerKey, void(*)(K &key, I const &inner) innerKey>
class ShuffleJoinRDD< O, I, K, outerKey, innerKey >

Join two RDDs using shuffle join. This join method is used when estimated result size is larger than inmemJoinThreshold. In this case inner and outer tables are shuffled into N files, where N=estimation/inmemJoinThreshold. The inner table files are one-by-one loaded in memory (placed in hash table and records frim corresponding outer table files are located in this hash table.
